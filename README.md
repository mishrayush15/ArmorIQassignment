### Note: 
Kindly use https://armoriqassignment-bxjp.onrender.com/docs or /docs endpoint for a better control over this project.


## Features

- /health - to check server health
- /docs - to use FastAPI GUI
- /create-account - to create new user account
- /deposit - to deposit money into an user's account
- /withdraw - to withdraw money from an user's account
- /balance - to check balance of an user's account
- /transactions - to view the transaction history of an user's account
